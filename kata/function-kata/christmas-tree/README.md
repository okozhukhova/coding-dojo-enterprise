# Christmas Tree
Write a function that outputs a christmas tree in ASCII art. 

**Input:**
> Height of the tree

`christmas-tree 5`

**Output:**
> The ASCII tree to terminal like this:
```
     X
    XXX
   XXXXX
  XXXXXXX
 XXXXXXXXX
     I
```
